# 游戏配置文件说明

本目录包含游戏的所有配置文件，支持热更新和参数调整。

## 配置文件列表

### 连连看模块
- `match3_config.json` - 连连看基础配置
- `match3_drop_config.json` - 资源掉落配置

### 塔防模块  
- `tower_config.json` - 弓箭塔属性配置
- `monster_config.json` - 怪物属性配置
- `combat_config.json` - 战斗系统配置

### 资源经济系统
- `resource_config.json` - 资源类型和数值配置
- `shop_config.json` - 商店配置

### NPC系统
- `dwarf_config.json` - 矮人NPC行为配置

### 关卡系统
- `level_config.json` - 关卡配置
- `wave_config.json` - 怪物波次配置

### 动画系统
- `animation_config.json` - 帧动画配置

### 界面配置
- `ui_config.json` - UI布局和样式配置

## 配置文件格式

所有配置文件采用JSON格式，支持注释和嵌套结构。

## 使用说明

1. 修改配置文件后，游戏会自动重新加载
2. 配置文件支持热更新，无需重启游戏
3. 请保持JSON格式的正确性
4. 备份重要配置文件
